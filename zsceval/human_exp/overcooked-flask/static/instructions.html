Steps 2-4

Onboarding, EEG Reminder, Tutorial

Before the interactive session starts, review the controls, the three task modes, and the probe logic. The experimenter should finish EEG setup and baseline collection before pressing the button below.

Game Basics

The participant controls the chef with the gray hat. The task in this session uses the ttt layout and starts with a tomato soup order.

Controls

Use the arrow keys to move and the spacebar to interact.

Mode Overview

Mode A: Observe

  • Watch AI behavior without controlling a chef.
  • The screen freezes mid-trajectory.
  • Predict the highlighted AI's immediate next action.

Mode B: Collaborate

  • Play with the AI teammate.
  • Probe freezes are event-triggered during play.
  • Answer what the AI is about to do next, then resume.

Mode C: Replay + Annotation

  • Replay scenes from the recent collaborate block.
  • Annotate what the AI intended.
  • Explain why the moment felt surprising.
